- **bulky**: takes 2 slots
- **weightless**: takes 0 slots
- **bundle**: takes 1 slot but can fit more items in that slot (e.g. trail
  rations 3 per bundle)
- **faulty** items cost 1d4 silver & have a Fair (+0) chance to break after each
  use.
